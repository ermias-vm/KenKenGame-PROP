He creat la classe Partida
Decisions:
-identificadorPartida es String
-en la creacio d'una partida nova s'inicialitza temps a 0
-en carregar una partida temps = tempsguardat i dataInicial = dataActual

-el format de guardar una partida no acabada es:

identificadorPartida
identificadorUsuari
identificadorTauler
temps
mida n
x_0,0 .. x_0,n-1
:
x_n-1,0

on x son valors entre 0 i n sent 0 el valor no inicialitzat. Estan separats per espais en la fila

-el format de guardar una partida acabada es:

identificadorPartida
identificadorUsuari
identificadorTauler
temps
mida n
guardada

guardada == true si la partida s'ha guardat en algun moment previ a acabarla, false si no
